## <p align="center">Como o próprio nome está dizendo, este aplicativo tem como intuito oferecer um buscador de notícias global, isto é, oferece a possibilidade de buscá-las em qualquer idioma retornando resultados únicos com base no termo de pesquisa utilizado.</p>

##

### <h3 align="center">Funcionalidades:</h3>

<div>
    <p>
      Atualmente, há uma página/rota na qual todo o aplicativo gira em torno.
      Tendo o Vite como ambiente de desenvolvimento, o APP foi criado em React, sendo aplicado à cada componente e instância príncipios metodológicos bem estabelecidos no mercado, como: a Metodologia BEM, SOLID e DRY.
      Quanto aos principais componentes. A "navbar" pode assumir duas formas, cuja primeira se adapta a largura de tela até um determinado limite; A partir desse limite, esta se ransforma em um menu retido em cujo ícone - semelhante à um hambúrguer - o usuário deve clicar para expandí-lo.
      A barra de busca é capaz de procurar por resultados em qualquer idioma e/ou termo. Tal busca pode ser regulada por certos critérios, sendo esses pré-estabelecidos no momento.
      Ademais, há os componentes de carregamento e de exibição das notícias nos quais a funcionalidade não detem nenhum comportamento anormal com exceção da adaptabilidade às demais larguras de tela.
      Por fim, todos os componentes tem a plena capacidade de se ajustaram as mais diferentes telas, conferindo-lhes o status de aplicativo responsivo.
    </p>
</div>

<div>
    <p>
      Para o futuro, a implementação do sistema de login, bem como uma espécie de biblioteca pessoal de notícias, serão os próximos alvos de desenvolvimento para este projeto.
    </p>
</div>
